Quick Start
===
**treevar($myvar);**
**treevar($myvar,false);**
**treevar($myvar,true);**

Description
===
Simple module contains  function **treevar($var, $open_all)**,
for print on web browser structure of variable in convenient
open-close structure.
**$open_all** = true - if you want watch open structure.
